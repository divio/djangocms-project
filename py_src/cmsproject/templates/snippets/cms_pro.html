Redaktion

Das «CMS Pro» ermöglicht, die internen redaktionellen Abläufe bei der Administration und beim Publishing der Website abzubilden. Dies beinhaltet die Bearbeitung durch mehrere Benutzer sowie die Freigabe von Inhalten durch definierte Moderatoren.

Rechte-Management

Den Benutzern können spezifische Rechte zugewiesen werden. Dies ermöglicht, dass bestimmte Seitenbereiche von den verantwortlichen Mitarbeitenden bearbeitet werden können.

Versionierung

Jede Modifikation einer Seite wird als separate Version gespeichert. Es ist jederzeit möglich, auf ältere Versionen einer Seite zuzugreifen und diese wiederherzustellen.

Multisites

Das CMS Pro erlaubt die Verwaltung mehrerer Websites über die gleiche Administration. So können beispielsweise verschiedene Länderwebsites zentral verwaltet werden.

Media Asset Manager (MAM)

Der MAM erlaubt die effiziente Verwaltung von Assets (Bildern, PDFs, Videos und weitere Dokumente). Assets, die in mehreren Seiten vorkommen, können mit dieser Verwaltung zentral administriert werden.
