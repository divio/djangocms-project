Flexible Plugin-Architektur

Um eine Seite anzulegen, können Sie aus verschiedenen Seitenvorlagen (Templates) auswählen und deren Inhalte anschliessend flexibel mit beliebigen Elementen (Plugins) gestalten.

Text Plugin

Anhand eines WYSIWYG-Editors bearbeiten Sie einfach selber alle Textinhalte ihrer Website. Dank WYSIWYG (What You See Is What You Get) sehen Sie sogleich wie Ihr Text auf der Website dargestellt wird. Die Bedienung erfordert keine speziellen HTML-Kenntnisse. Fortgeschrittene können HTML-Inhalte direkt einfügen.

Picture Plugin

Mit dem Picture Plugin fügen Sie Bildinhalte ein, ohne sich dabei um Grössen zu kümmern: Das Plugin passt die Bilder automatisch auf die vorgegebene Grösse an.

Suchmaschinenoptimierung (SEO)

Divio erstellt alle Websites nach den Vorgaben der W3C (World Wide Web Consortium) und optimiert die Struktur der Website, damit Ihre Website auch einfach von den Suchmaschinen indexiert werden kann.

Zugriffsstatistik

Mit unserem CMS Starter konnen Sie uber Google Analytics einfach verfolgen, wie sich die Besucherzahlen auf Ihrer Website entwickeln. Dank der Anbindung an Ihren Google Analytics Account erhalten Sie statistische Daten uber Besuchertrends, Seitenaufrufe, Zugriffsquellen, Verweildauer und viele weitere Auswertungen.
